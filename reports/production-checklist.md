## Phase 3: UGC Content (Week 3+)

### UGC Testimonial

**Sourcing:**
- [ ] Reach out to top reviewers for video testimonials
- [ ] OR hire UGC creators (specify brief)
- [ ] OR source from existing customer videos

**Brief Requirements:**
- [ ] Vertical format (9:16)
- [ ] Natural lighting (window light preferred)
- [ ] Casual setting (home, kitchen)
- [ ] Authentic, unscripted feel
- [ ] Must include:
  - [ ] Initial skepticism
  - [ ] First experience
  - [ ] Transformation/results
  - [ ] Recommendation
- [ ] Product must be visible
- [ ] 30-60 seconds length

**Post-Production:**
- [ ] Light editing only (keep authentic)
- [ ] Add captions
- [ ] Add subtle branding (end card only)
- [ ] Export for Reels/Stories

## Technical Specifications

### Image Ads
| Spec | Requirement |
|------|-------------|
| Format | JPG or PNG |
| Resolution | 1080px minimum width |
| File size | <30MB |
| Text coverage | <20% of image area |
| Safe zone | Keep key elements 14% from edges |

### Video Ads
| Spec | Requirement |
|------|-------------|
| Format | MP4 (H.264) |
| Resolution | 1080p minimum |
| Frame rate | 30fps |
| Duration | 15-60 seconds |
| Aspect ratio | 9:16 (primary), 4:5 (secondary) |
| File size | <4GB |
| Captions | Required (SRT or burned in) |
| Audio | -14 LUFS (normalized) |

## Brand Guidelines Quick Reference

### Colors
- **Primary Navy:** #1a365d
- **Accent Blue:** #3182ce
- **Success Green:** #38a169
- **Background:** #f7fafc

### Typography
- **Headlines:** Bold, clean sans-serif
- **Body:** Regular weight, readable at small sizes
- **Avoid:** Script fonts, decorative fonts

### Logo Usage
- Place in corner (bottom right preferred)
- Keep subtle — product is hero
- White version on dark backgrounds

---

## Quality Checklist (All Creatives)

- [ ] Mobile-first: Looks good on phone screen
- [ ] Hook in 1 second: Attention grabbed immediately
- [ ] Text readable: Even at thumbnail size
- [ ] Brand consistent: Colors, fonts, tone
- [ ] CTA clear: Viewer knows what to do
- [ ] Compliant: No prohibited claims
- [ ] Tested: Shown to someone unfamiliar for feedback
